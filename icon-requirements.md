# Icon Requirements

The extension needs the following icon files in PNG format:

## Required Icons

### icon16.png
- Size: 16x16 pixels
- Usage: Extension toolbar icon (small)
- Suggested design: Simple brain/filter icon in LinkedIn blue (#0073b1)

### icon48.png  
- Size: 48x48 pixels
- Usage: Extension management page
- Suggested design: Detailed brain/filter icon with LinkedIn styling

### icon128.png
- Size: 128x128 pixels  
- Usage: Chrome Web Store and extension details
- Suggested design: High-resolution version with gradient and effects

## Creating Icons

You can create these icons using:
- **Free tools**: GIMP, Paint.NET, Canva
- **Online tools**: Favicon.io, IconMaker
- **AI tools**: DALL-E, Midjourney for initial designs
- **Design software**: Photoshop, Illustrator, Figma

## Temporary Solution

For testing purposes, you can:
1. Create simple colored squares with text
2. Use online icon generators
3. Convert existing emoji to PNG format
4. Use placeholder icons from icon libraries

Once you have the PNG files, place them in the root of the extension folder.
